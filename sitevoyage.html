Site d'articles de voyage

Site PHP

Nous avions décidé de créer un site permettant de publier des articles sur nos voyages. Les personnes peuvent se connecter et créer leurs propres articles. Nous avons utilisé du PHP, CSS et SQL. Nous avons créé des bases de données afin de permettre à l'utilisateur de se créer un compte et de s'identifier par la suite. Ces bases de données nous ont aussi servi à stocker les articles et toutes les images du site. Nous nous sommes aidés de Bootstrap ainsi que de Templated pour réaliser le CSS du site.
