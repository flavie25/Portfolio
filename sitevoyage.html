Site d'articles de voyage

Site PHP

Nous avions décidé de créer un site permettant de publié des articles sur nos voyages. Les personnes peuvent se connecter et créer leur propre articles. Nous avons utiliser du PHP, CSS et SQL. Nous avons créer des bases de données afin de permettre à l'utilisateur de se créer un compte et de s'identifier par la suite. Ces bases de données nous on aussi servis à stocker les articles et toutes les images du site. Nous nous sommes aidé de Bootstrap ansi que de Templated pour réaliser le css du site.
